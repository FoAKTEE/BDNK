## 1. IS / MIS setup (Maxwell-Cattaneo constitutive relation)

Energy-momentum tensor (Eckart frame), only bulk viscosity (no heat conduction, no shear) — Eq.(eq:energy_momentum):
- `T^{μν} = e u^μ u^ν + (p+Π) h^{μν}`
- `J^μ = ρ u^μ`

**Maxwell-Cattaneo (Israel-Stewart) relaxation equation** — Eqs.(eq:is_bulk),(eq:nsvalue):
- `τ_Π u^μ ∇_μ Π = Π_NS − Π`
- `Π_NS := −ζ Θ`  (Navier-Stokes value)
- `Θ := ∇_μ u^μ` (fluid expansion). 3+1 form (eq:3+1expansion): `Θ = ϑ + Λ − KW`.

Lorentz factor `W := (1 − v_i v^i)^{-1/2}`; coordinate velocity `V^j := u^j/u^t = α v^j − β^j`.

**Inverse Reynolds number** (Denicol2012b) — Eq.(eq:reynolds):
- `R^{-1} := Π/(p+e)`

The bulk-viscous pressure Π is evolved as an **independent variable** (the conserved variable is `DΠ = ρWΠ`), see state/flux/source vectors Eqs.(eq:bulk_implement1)-(eq:bulk_implement2). Source term for DΠ:
`−(αD/(τ_Π W))[ζ(ϑ + Λ − KW) + Π]`.

Code: **FIL** (4th-order finite differences, Z4c spacetime, ECHO scheme built on IllinoisGRMHD). Time derivative of Π source: 1st-order backward differencing wrt previous full timestep; spatial via 4th-order central FD.

Atmosphere value: `Π_atmo = 0`.

## 2. Causality-fixed sound speed equations (KEY for "causality-fixed sound speed")

**Viscous sound speed** (Bemfica2019, Chabanov2021, Camelio2022_a), the square of the sound speed including bulk-viscous modifications:

```
c_s'^2 = (ζ/τ_Π)·(1/(ρ h')) + (∂p/∂e)_ρ + (1/h')·(∂p/∂ρ)_e
```

with `h' := h + Π/ρ`, `h := (e+p)/ρ`, `a' := (p+Π)/(ρ(1+ε))`.

**Limiting requirements** (dominant energy condition + subluminal sound speed) — Eqs.(eq:limit_aprime),(eq:limit_sound):
- `0 ≤ a' ≤ 1`
- `0 ≤ c_s'^2 < 1`

**Limiting strategy** (Section sec:limiting):
- (i) If `Π < 0` and `Π < σ p` with `−1 ≤ σ ≤ 0`, set `Π = σ p`. **Typical choice `σ = −0.9`** (avoids pressureless regions p+Π=0).
- (ii) If `Π > 0` and `a' > 1`, set `Π = e − p`.
- (iii) **Causality fix** — if `c_s'^2 > c_max^2`, reset τ_Π via Eq.(eq:causal_limit):

```
τ_Π = (ζ/(ρ h')) · [ c_max^2 − (∂p/∂e)_ρ − (1/h')(∂p/∂ρ)_e ]^{-1}
```

with `c_max` a free parameter, `0 ≤ c_max^2 < 1`, explicitly enforcing causality (locally increases τ_Π).

**Existence/uniqueness B factors** (root-finding `f(z) = z − r/h'(z)`, `df/dz = 1 − v²B`):
- Π constant: `B_Π = (∂p/∂e)_ρ + (1/h')(∂p/∂ρ)_e ≤ c_s'^2 < 1`
- Π limited from below: `B_(i) = (1−σ){ (∂p/∂e)_ρ + (ρ/((1−σ)p+e))(∂p/∂ρ)_e } ≤ c_s^2 < 1`
- Π limited from above: `B_(ii) = a' = 1`

Bracketing interval (unchanged): `z_- = (k/2)/√(1−k²/4)`, `z_+ = k/√(1−k²)`, with `k(v,a') = v(1+a')/(1+v²a')`.

## 3. Reynolds number limits (ideal-fluid EOS estimate)

EOS `p = (Γ_id − 1)ρε`, isentropic `p = κ ρ^{Γ_id}`.

- `R^{-1}_min = σp/(e+p) = σ / (κ^{-1}ρ^{1−Γ_id} + 1 + (Γ_id−1)^{-1})`
- `R^{-1}_max = (e−p)/(e+p) = ((Γ_id−1)ρ^{1−Γ_id}/κ − Γ_id + 2)/((Γ_id−1)ρ^{1−Γ_id}/κ + Γ_id)`

**Verbatim numeric targets** (conditions: `Γ_id = 2`, `κ = 100`, `ρ = 0.00128`, `σ = −0.9`):
- `R^{-1}_min ≃ −0.1`
- `R^{-1}_max ≃ 0.8`

## 4. Transition zone (ζ and τ_Π density dependence) — Eq.(eq:zeta_bound)

```
ζ(ρ) = ζ_l                        for ρ ≤ ρ_l
       a ρ³ + b ρ² + c ρ + d      for ρ_l < ρ < ρ_h   (cubic, C¹ at both ends)
       ζ_h                        for ρ ≥ ρ_h
```
Coefficients:
- `a = 2(ζ_l−ζ_h)/(ρ_h−ρ_l)³`
- `b = 3(ρ_l+ρ_h)(ζ_h−ζ_l)/(ρ_h−ρ_l)³`
- `c = 6ρ_h ρ_l(ζ_l−ζ_h)/(ρ_h−ρ_l)³`
- `d = [(3ρ_h ρ_l² − ρ_l³)ζ_h + (ρ_h³ − 3ρ_h²ρ_l)ζ_l]/(ρ_h−ρ_l)³`

Relaxation time (linear interp in transition zone):
```
τ_Π(ρ) = τ_l                      for ρ ≤ ρ_l
         τ_h − m(ρ_h − ρ)         for ρ_l < ρ < ρ_h
         τ_h                      for ρ ≥ ρ_h
```
with `m = (τ_h − τ_l)/(ρ_h − ρ_l)`. Typical: `τ_h ∼ 1.1 Δt_min`, `τ_l ∼ 1.1 Δt_max`.

- **Mass** `M = 1.447 M_⊙`

**Migration outcome** (the bulk-viscous migration/collapse result):
- The unstable star migrates to the stable branch with damped nonlinear central-density oscillations; **higher ζ → shorter damping time** (oscillations of ρ_c decay more efficiently). No collapse to BH — migration to stable branch.
- **Average central density around which oscillations occur DECREASES with increasing ζ** (more thermal support → less compact asymptotic state, smaller central ρ).
- Thermal-pressure ratio `p_th/p` (with `p_th := (Γ_th−1)ρε_th`): higher ζ → faster initial rise + higher asymptotic value. **For highest-viscosity case p_th can reach `∼10%` of total pressure** (stated in Conclusions: "up to ∼10% of the total pressure for the highest viscosity case"). [Exact curves in Fig.~mig_central middle panel — figure only.]
- Inverse Reynolds number `|R^{-1}|` in center: **dramatic viscous influence on ρ_c evolution for flows with `|R^{-1}| ≳ 10^{-2}`**; simulations remain within domain of applicability (low R^{-1}). Good agreement between true Π and Π_NS (only weak deviation from first-order theory while staying causal). NS values shown only for high-visc case in Fig.~mig_central right panel.

**Shock propagation** (Fig.~mig_shocks):
- Density contours plotted at `ρ = 4.5×[10^{14},10^{12},10^{10},10^{8}] g cm^{-3}`. Inner solid line = ρ_h, outer solid line = ρ_l.
- Larger ζ → shock develops at larger distance from core; weaker (slower) shocks for 2nd/3rd waves.
- **In the high-viscosity case the 2nd-wave shock front STOPS propagating when it reaches `∼60 km`.** [from text; spacetime diagram Fig.~mig_shocks]

- `ρ_h ≈ 4.52×10^{14} g cm^{-3}`

**Resolution:** `Δx ∼ 0.17 M_⊙ ≈ 260 m` (6th refinement level); lower-res checks `Δx ∼ 0.25 M_⊙ ≈ 370 m`.

**Diagnostic integrals:**
- Thermal energy `E_th := ∫ ρ ε W √γ dx dy dz` (Eq.(eq:en_thermal))
- z-kinetic energy `E_z := (1/2)∫ ρ h v^z u_z √γ dx dy dz` (Eq.(eq:en_z))
- m=2 bar mode `P_m := ∫ ρ W e^{-imφ} √γ dx dy dz`, m=2 (Eq.(eq:mode))

**Key BNS numeric results (verbatim):**
- Strong damping of density oscillations within first `∼5 ms` after merger.
- High-viscosity inverse Reynolds number reaches `> ∼1%`, corresponding to bulk-viscous pressure `∼20%` of equilibrium/EOS pressure; this peak is right after merger and persists until `t−t_mer ≈ 1.5 ms`.
- After sharp drop (~2 orders of magnitude): at `t−t_mer ≈ 5 ms`, `(|Π|/ρh)_c ≈ 10^{-5}` and `(|Π|/p)_c ≈ 10^{-4}`.
- Snapshots Fig.~bin_temp_pip at `t−t_mer = 4.22 ms`; Fig.~rho_temp_xz at `t−t_mer = 23 ms`; Fig.~bin_bernoulli at `t−t_mer = 7.11 ms`.
- "Cold" material threshold `T ≲ 5 MeV`.
- **Dynamical ejecta: high-viscosity total ejecta mass is only `∼20%` of zero-viscosity value (suppression factor `∼5`).**
- Ejecta velocity: large ζ suppresses ejecta with `v ≲ 0.2` and `v ≳ 0.5`; median shifts higher.
- Mollweide contour level `8×10^{-7} M_⊙^{-2}`; enhanced ejection at `φ ≈ π/4` and `φ ≈ 5π/4`.
- Detector at `∼517 km`; GW extracted at `∼740 km`, normalized to 100 Mpc.
- Abstract: inverse Reynolds numbers `∼1%`; ejected mass suppressed by `∼5`.
- HMNS end time `t−t_mer ≈ 24 ms`; rotational profiles at `∼23 ms`, angular velocity peak at `≃4–5 km`, core `r ≲ 8 km`.

## 9. Reproduction notes / convention summary

- **Central density convention:** ρ_c is the **baryonic rest-mass density** at the stellar center, in geometrised `M_⊙^{-2}` (with cgs conversions given). Conversion used by authors: `1.28×10^{-3} M_⊙^{-2} ≈ 7.91×10^{14} g cm^{-3}` (i.e. `1 M_⊙^{-2} ≈ 6.18×10^{17} g cm^{-3}`).
- **Eckart frame** (not Landau). Π evolved as independent variable (true second-order IS, not Navier-Stokes), but Π ≈ Π_NS = −ζΘ in practice (near first-order regime, but causal).
- The "causality fix" is the limiter (iii) that dynamically rescales τ_Π so that c_s'^2 ≤ c_max^2 (Eq.(eq:causal_limit)).
- Units of ζ throughout: `g s^{-1} cm^{-1}` (= g cm^{-1} s^{-1}). τ in ms or geometrised.
