## Nature of the paper (IMPORTANT for "targets")

This is a **purely analytic stability/causality paper**. It contains:
- **No equation of state (EOS) numerics** (no polytrope, no tabulated EOS). The only EOS input is the speed of sound `v_s^2 = dp/dε`, treated as a free parameter (`0 < v_s < 1`).
- **No neutron-star tables, no M-R curves, no numeric tables at all.** Every "number" is either (a) an analytic inequality / stability condition, or (b) an illustrative parameter choice used to render a figure.
- The transport coefficients are the *general-frame* one-derivative coefficients ε_i, π_i, θ_i, ν_i, γ_i, η (not numeric values — they are the free parameters whose *allowed regions* are the result).

So the "exact reproducible targets" are: (1) the general-frame constitutive relations [Eq. (2.7)/(2.14)], (2) the frame-transformation rules and frame invariants, (3) the **shear-channel** stability condition, (4) the four **sound-channel** stability/causality inequalities, and (5) the **conformal** special-case conditions. All have exact equation numbers. Figure parameter choices are recorded separately with explicit "figure-only" notes.

Equation numbering uses `\numberwithin{equation}{section}`, so labels are `(section.number)`. Section 2 = Constitutive relations, Section 3 = Stability, Section 4 = Uncharged fluids.

## 1. General-frame first-order constitutive relations (THE core result)

### Decomposition [Eq. (2.2) = `eq:T1`, Eq. (2.3) = `eq:EPNqjt`]
Given a timelike unit vector u^μ, with Δ^{αβ} ≡ g^{αβ} + u^α u^β (mostly-plus signature; u² = −1):
- T^{μν} = E u^μ u^ν + P Δ^{μν} + (Q^μ u^ν + Q^ν u^μ) + T^{μν}_{tt}
- J^μ = N u^μ + J^μ_⊥
- E ≡ u_μ u_ν T^{μν}, P ≡ (1/d) Δ_{μν} T^{μν}, Q_μ ≡ −Δ_{μα} u_β T^{αβ}, N ≡ −u_μ J^μ.

### General-frame constitutive relations [Eq. (2.7) = `eq:EPQTNJ`] — the central object
Dot = u^λ ∂_λ. d spatial dimensions. σ^{μν} is the transverse-traceless shear tensor.
- E   = ε + ε₁ Ṫ/T + ε₂ ∂·u + ε₃ u^λ ∂_λ(μ/T) + O(∂²)
- P   = p + π₁ Ṫ/T + π₂ ∂·u + π₃ u^λ ∂_λ(μ/T) + O(∂²)
- Q^μ = θ₁ u̇^μ + (θ₂/T) Δ^{μλ}∂_λ T + θ₃ Δ^{μλ}∂_λ(μ/T) + O(∂²)
- T^{μν}_{tt} = −η σ^{μν} + O(∂²)
- N   = n + ν₁ Ṫ/T + ν₂ ∂·u + ν₃ u^λ ∂_λ(μ/T) + O(∂²)
- J^μ_⊥ = γ₁ u̇^μ + (γ₂/T) Δ^{μλ}∂_λ T + γ₃ Δ^{μλ}∂_λ(μ/T) + O(∂²)

**Coefficient count:** 16 one-derivative coefficients for charged fluids (ε₁₂₃, π₁₂₃, θ₁₂₃, ν₁₂₃, γ₁₂₃, η); **7 for uncharged** (ε₁₂, π₁₂, θ₁₂, η — γ and ν drop out). Zero-derivative: ε, p, n.

### Field-redefinition ("frame change") rules [Eq. (2.10) = `eq:frame-change-3`]
Under δT = a_i(...), δu^μ = b_i(...), δμ = c_i(...) [Eq. (2.9) = `eq:frame-change-2`]:
- ε_i → ε_i − ε_{,T} a_i − ε_{,μ} c_i
- π_i → π_i − p_{,T} a_i − p_{,μ} c_i
- ν_i → ν_i − n_{,T} a_i − n_{,μ} c_i
- θ_i → θ_i − (ε+p) b_i
- γ_i → γ_i − n b_i
- η  → η

### Frame invariants [Eq. (2.11) = `eq:frame-invariants-1`]
- f_i ≡ π_i − (∂p/∂ε)_n ε_i − (∂p/∂n)_ε ν_i
- ℓ_i ≡ γ_i − [n/(ε+p)] θ_i
- plus η.
**Only 3 genuine one-derivative transport coefficients survive:** η, ζ (bulk), σ (charge conductivity).

### Extensivity / thermodynamic-consistency constraints [Sec. 2.2.1]
- ε = −p + T ∂p/∂T + μ ∂p/∂μ, n = ∂p/∂μ (zeroth order).
- **θ₁ = θ₂, γ₁ = γ₂ ⇒ ℓ₁ = ℓ₂** (one-derivative consistency). Hence χ_T = 0.

### Landau-Lifshitz frame [Eq. (2.14) = `eq:Landau-frame-2`], with [Eq. (2.15) = `eq:zetasigma`]
On-shell L-L constitutive relations reduce to η, ζ, σ. For uncharged fluid (footnote, `fn:zeta`):
**ζ = v_s²(π₁ − v_s² ε₁) − π₂ + v_s² ε₂**, where v_s² ≡ ∂p/∂ε. (Appendix gives equivalent ζ = −π₂ + v_s²(ε₂+π₁) − v_s⁴ε₁.)
- σ = [n/(ε+p)] ℓ₁ − (1/T) ℓ₃; χ_T = (1/T)(ℓ₂ − ℓ₁) = 0.

### Eckart frame [Eq. (2.16)=`eq:Eckart-frame-1`, (2.17)=`eq:QE`, (2.18)=`eq:QE-1`, (2.19)=`eq:QE-2`]
κ ≡ (ε+p)² σ /(n² T) (heat conductivity).

## 2. Stability & causality definitions [Sec. 3]
- **Stability** [Eq. (3.2) = `eq:Imwl0`]: **Im ω(k) ≤ 0 for all k.**
- **Causality** [Eq. (3.5) = `eq:cc`]: **lim_{k→∞} Re ω(k)/k < 1.**
- Eigenfrequencies from H(w,z)=0 [Eq. (3.6)], factorizing H = F_shear^{d-1} F_sound [Eq. (3.8) = `eq:H2`], with w ≡ −q·ū, z ≡ q² + (q·ū)², q^α=(ω,k).
- Boost-covariance relation between v₀=0 and v₀≠0 [Eq. (4.2) = `eq:FF-boosted`].
- Linear-mode velocity addition [Eq. (4.3) = `eq:cvphi`]: c_v(φ) = v₀(1−c₀²)/(1−c₀²v₀²) cosφ ± [c₀/(1−c₀²v₀²)]√{(1−v₀²)[1 − v₀²c₀² − v₀²(1−c₀²)cos²φ]}.

## 3. SHEAR-CHANNEL stability condition (EXACT TARGET)

Shear quadratic [Eq. (4.4) = `eq:shear-quadratic`]:
F_shear = (θ − v₀²η)ω² + (i w₀/γ₀ − 2(θ−η)k·v₀)ω − (i w₀/γ₀)(k·v₀) − k²η/γ₀² + (k·v₀)²(θ−η),
with γ₀ ≡ 1/√(1−v₀²), w₀ ≡ ε₀+p₀.

**Stability condition [Eq. (4.7) = `eq:shear-stability`]:**
> **θ > η > 0.**

Notes: θ ≡ θ₁ = θ₂. Stability depends on the single dimensionless ratio θ/η (ω, k measured in units of (ε₀+p₀)/η). Landau-Lifshitz sets θ=0 ⇒ unstable. Large-k shear speed satisfies Eq. (4.10)=`eq:cs-shear`, bounded |c_shear| < (1+|v₀|(θ/η)^{1/2})/(|v₀|+(θ/η)^{1/2}) < 1 for θ>η.

## 4. SOUND-CHANNEL stability/causality conditions (EXACT TARGETS)

Sound quartic at v₀=0 [Eq. (4.11) = `eq:Fsound0`]:
F_sound = v_s²ε₁θ ω⁴ + i w₀(v_s²ε₁+θ)ω³ − [w₀² + k²v_s²(v_s⁴ε₁² + γ_s ε₁ + (ε₂+π₁)(θ−v_s²ε₁) + ε₂π₁)]ω² − i k²w₀(γ_s + v_s⁴ε₁ + v_s²θ)ω + k²v_s²[w₀² + k²θ(v_s²(ε₂+π₁−v_s²ε₁) − γ_s)].
Definitions: **v_s² ≡ ∂p₀/∂ε₀**, **γ_s ≡ (4/3)η + ζ** (sound damping), w₀ ≡ ε₀+p₀.
π₂ traded for ζ via footnote `fn:zeta`. Stability governed by 4 dimensionless params: ε₁/γ_s, ε₂/γ_s, π₁/γ_s, θ/γ_s.

### Small-k necessary conditions [from Eq. (4.13)]:
> **γ_s > 0, ε₁ > 0, θ > 0.**

Gapped frequencies at v₀=0: ω = −i(ε₀+p₀)/(v_s²ε₁) and ω = −i(ε₀+p₀)/θ [Eq. (4.13)=`eq:wgaps-2`].

### Routh-Hurwitz conditions at arbitrary k, v₀=0 [Eqs. (4.15)-(4.17)]:
Set ω = iΔ → quartic aΔ⁴+bΔ³+cΔ²+dΔ+e=0 with a>0,b>0,d>0. Stability ⇔ e>0 AND e/a < (d/b)(c/a − d/b).

**Condition 1 [Eq. (4.16) = `eq:sound-c1`]:**
> **ε₂ + π₁ > γ_s/v_s² + v_s² ε₁.**

**Condition 2 [Eq. (4.17) = `eq:sound-c2`]** (nonlinear, in dimensionless bars):
> (ε̄₁²/v_s²) + v_s²(ε̄₁−ε̄₂)(ε̄₁+θ̄)²(ε̄₁−π̄₁) + (ε̄₁+θ̄)[2ε̄₁² − ε̄₁(ε̄₂+π̄₁) + (θ̄+ε̄₂)(θ̄+π̄₁)] > 0,

with **ε̄₁ ≡ v_s²ε₁/γ_s, ε̄₂ ≡ ε₂/γ_s, θ̄ ≡ θ/γ_s, π̄₁ ≡ π₁/γ_s.**

### Large-k causality at v₀=0 [Eq. (4.18) = `eq:cs0largek`]:
ω = c_sound k with
(ε̄₁θ̄/v_s²) c_sound⁴ + [ε̄₁(ε̄₂+π̄₁−ε̄₁−1/v_s²) − θ̄(ε̄₂+π̄₁) − ε̄₂π̄₁] c_sound² + θ̄[v_s²(ε̄₂+π̄₁−ε̄₁) − 1] = 0.
Require c_sound² real, positive (stability) and <1 (causality). Causality footnote: for ax²+bx+c=0 (a>0), 0<x<1 ⇔ b²>4ac, 0<c<a, −c−a<b<0.

### Moving-fluid gapped-mode necessary condition [Eq. (4.22)]:
> **v_s² ε₁ + θ > γ_s/(1−v_s²).**

This excludes a finite neighbourhood of ε₁=θ=0 for any η>0. Landau-Lifshitz (ε₁=θ=0) ⇒ unstable at v₀≠0.

### Discussion-section summary of necessary stability conditions [Sec. 5]:
> θ > η > 0;  v_s² ε₁ + θ > γ_s/(1−v_s²);  ε₂ + π₁ > γ_s/v_s² + v_s² ε₁.

## 5. CONFORMAL (uncharged, 3+1) special case [Sec. 4.2 / Eqs. (4.23)-(4.24)]

Conformal relations [Eq. (4.23) = `eq:cft-relations`]: ε₁ = 3π₁, ε₂ = 3π₂, π₁ = 3π₂. Speed of sound **v_s = 1/√3**, bulk viscosity ζ = 0. Three independent coefficients: π₁, θ, η.

**Stability conditions [Eq. (4.24) = `eq:cft-stable`]:**
> **1 − 3η/θ − η/π₁ > 0   AND   π₁ > 4η.**

Agrees with Bemfica-Disconzi-Noronha (arXiv:1708.06255 = ref [Bemfica:2017wps]). Sufficient simple choice: **θ > 4η, π₁ > 4η** ⇒ stable + causal.

## 7. Method to reproduce the analytic targets

1. Build the general-frame T^{μν}, J^μ from Eq. (2.7). For uncharged fluid drop ν,γ; impose θ₁=θ₂≡θ.
2. Linearize ∂_μT^{μν}=0 about (T₀, v₀) with plane waves e^{ik·x − iωt}. Split into shear (δv⊥k) and sound (δv∥k, δT) channels.
3. **Shear:** form the 2×2 determinant → quadratic Eq. (4.4); set ω=iΔ; require Im ω ≤ 0 ⇒ recover θ>η>0 [Eq. (4.7)].
4. **Sound at v₀=0:** form quartic Eq. (4.11); substitute ω=iΔ → real-coefficient quartic; apply Routh-Hurwitz (a,b,d>0, e>0, and e/a < (d/b)(c/a−d/b)) ⇒ Eqs. (4.16),(4.17). Large-k: drop lower orders → biquadratic Eq. (4.18); demand 0<c_sound²<1.
5. **Moving fluid:** apply boost map Eq. (4.2) to F_sound(v₀=0); the gap-stability necessary condition is Eq. (4.22).
6. **Conformal check:** set v_s²=1/3, ε₁=3π₁, ζ=0; verify Eq. (4.24) and the BDN agreement.
7. To regenerate Fig. 2: scan (θ̄, ε̄₁) with ε₂=0, π₁/γ_s=3/v_s², shade where Eqs. (4.16)+(4.17) (left) and additionally Eq. (4.18) causality (right) hold, for several v_s.

No ODE/EOS integration is needed; everything is polynomial root analysis (Routh-Hurwitz on quartics) and algebraic inequalities.
