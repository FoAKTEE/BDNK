## 1. EOS, conformal frame, and transport coefficients

### Conformal closure (eq. block at line 425, the "eqs 19–20" region)
For a conformal fluid with `μ = 0` (`g_{ab}T^{ab}=0`):
```
P(ε,n) = ε/3,   Π = A/3,   ζ = 0,
β_ε = τ_Q/3,    β_n = 0,    τ_P = τ_ε/3.
```
This leaves only `η`, `τ_ε`, `τ_Q` as independent transport coefficients; `T^{ab}`
becomes independent of `n`, so `J^a` and `T^{ab}` **decouple**.

Thermal transport coefficients (general, eq. block line 407 — eqs 18–19 in arabic
numbering):
```
β_ε = τ_Q (∂P/∂ε)_n + [σ T (ε+P)/n] (∂(μ/T)/∂ε)_n
β_n = τ_Q (∂P/∂n)_ε + [σ T (ε+P)/n] (∂(μ/T)/∂n)_ε
```

### Frame-coefficient definitions — `eq:frame_coeffs_def` (line 441; arabic eq. 21)
The three remaining coefficients are written with the ε-dependence pulled out:
```
η     ≡ η_0 ε^{3/4}
τ_ε = (3/4ε) χ ≡ (3/4ε) χ_0 ε^{3/4}
τ_Q = (3/4ε) λ ≡ (3/4ε) λ_0 ε^{3/4}
```
`η_0` is the dimensionful free parameter controlling viscosity; `(χ_0, λ_0)` set
the hydrodynamic frame.

### Frame choice — `eq:frame_coeffs` (line 455; arabic eq. 22)  [CRITICAL constant]
Following Pandya 2021:
```
(χ_0, λ_0) = ( (25/4) η_0 , (25/7) η_0 ).
```
This choice **fixes the characteristic speeds to be exactly unity** (= speed of
light). Consequence used throughout: the Kurganov–Tadmor max local speed `a = 1`
is *exact*, not tuned. Inviscid limit `η_0 → 0` ⇒ `χ_0, λ_0 → 0` ⇒ recovers the
perfect-fluid currents exactly.

## 2. Conformal-frame coordinate coefficients of the BDNK currents (Appendix A)

Conserved variables in conservative form (`eq:delT_conserv_form`, line 612;
`eq:delJ_conserv_form`, line 641):
```
q  = (T^{tt}, T^{tx}, T^{ty})^T,  f^x = (T^{tx}, T^{xx}, T^{yx})^T,
f^y = (T^{ty}, T^{xy}, T^{yy})^T,  ψ = 0
q = J^t, f^x = J^x, f^y = J^y, ψ = 0
```

Primitive variables: PF `p_0 = (ε, u^x, u^y)^T`; BDNK `p_1 = (ξ̇, u̇^x, u̇^y)^T`
with `ξ ≡ ln(ε)` (evolved instead of ε for stability), plus trivial evolution
equations `∂_t ξ = ξ̇`, `∂_t u^x = u̇^x`, `∂_t u^y = u̇^y` (`eq:trivial_evol_eqns`).

Four-velocity: `u^a = (√(1+(u^x)²+(u^y)²), u^x, u^y, 0)^T`.

### Energy-density correction A (line 1646)
```
A = (1/4) χ_0 e^{(3/4)ξ} [ u^x(4 u̇^x/u^t + 3 ξ_{,x}) + u^y(4 u̇^y/u^t + 3 ξ_{,y})
                          + 3 ξ̇ u^t + 4 u^x_{,x} + 4 u^y_{,y} ]
```
(Note `Π = A/3` by conformality.)

### Heat-flux components (lines 1651, 1656)
```
Q^x = (1/4) λ_0 e^{(3/4)ξ} [ 4 u̇^x u^t + u^x ξ̇ u^t + ((u^x)²+1) ξ_{,x}
                            + 4 u^x u^x_{,x} + u^y(u^x ξ_{,y} + 4 u^x_{,y}) ]
Q^y = (1/4) λ_0 e^{(3/4)ξ} [ u^y(u^t ξ̇ + u^x ξ_{,x} + u^y ξ_{,y} + 4 u^y_{,y})
                            + 4 u^t u̇^y + 4 u^x u^y_{,x} + ξ_{,y} ]
Q^t = (1/u^t)(u^x Q^x + u^y Q^y),   Q^z = 0   (from u_a Q^a = 0)
```

### Shear components −2η σ^{ab} (lines 1667, 1673, 1679)
```
-2η σ^{xx} = (2 η_0 e^{(3/4)ξ})/(3 u^t) [ u^t(-2((u^x)²+1)u^x_{,x} + (u^x)² u^y_{,y}
             - 3 u^x u^x_{,y} u^y + u^y_{,y}) - 2(u^x)³ u̇^x + (u^x)² u^y u̇^y
             - u^x u̇^x(3(u^y)²+2) + u^y u̇^y ]

-2η σ^{xy} = -(η_0 e^{(3/4)ξ})/(3 u^t) [ u^t(3(u^x)² u^y_{,x} + u^x u^y(u^x_{,x}+u^y_{,y})
             + 3 u^x_{,y}((u^y)²+1) + 3 u^y_{,x}) + u̇^x u^y((u^x)²+3(u^y)²+3)
             + u^x u̇^y(3(u^x)²+(u^y)²+3) ]

-2η σ^{yy} = (2 η_0 e^{(3/4)ξ})/(3 u^t) [ -3 u^t u^x u^y u^y_{,x} + u^t u^x_{,x}((u^y)²+1)
             - 2 u^t((u^y)²+1)u^y_{,y} - 3(u^x)² u^y u̇^y + u^x u̇^x((u^y)²+1)
             - 2((u^y)³+u^y)u̇^y ]

σ^{tc} = (1/u^t)(u^x σ^{xc} + u^y σ^{yc})   (from u_a σ^{ab} = 0)
```
For 1D problems (Gaussian, shock tube, steady-state shock): `T^{cy}_1 = u^y = 0`.

## 3. Primitive variable recovery (Sec. III C)

### Perfect-fluid analytic PVR — `eq:PF_pvr` (line 669)  [exact, conformal]
```
ε   = -T^{tt} + √( 6(T^{tt})² + 3[(T^{tt})² - (T^{tx})² - (T^{ty})²] )
|v| = √((T^{tx})²+(T^{ty})²) / (T^{tt} + 3ε),   u^t = 1/√(1-|v|²)
u^x = 3 u^t T^{tx}/(3 T^{tt}+ε),   u^y = 3 u^t T^{ty}/(3 T^{tt}+ε)
```

### BDNK PVR linearity — `eq:BDNK_linearity_in_p` / `eq:naive_BDNK_pvr`
`q_1 = q_0(p_0) + η_0 [ A(p_0)·p_1 + b(p_0, ∂_i p_0) ]`, so
`p_1 = A^{-1}·[ (1/η_0)(q_1 − q_0) − b ]`. `A^{-1}` always exists in the chosen
frame for physical states.

### Shifted/adaptive BDNK PVR — `eq:q_tilde_defn`, `eq:shifted_BDNK_pvr`
```
q̃_1 ≡ q_1 − q_1|_{p_1→p_1^PF} = η_0 A·(p_1 − p_1^PF)
p_1 = (1/η_0) A^{-1}·q̃_1 + p_1^PF
```
Adaptive algorithm (line 763): compute numerical-viscosity estimate → viscous
tolerance `Δ_η`; compute `q̃_1`; if `q̃_1 ≥ Δ_η` use shifted PVR + trivial
evolution; if `q̃_1 < Δ_η` set `q̃_1 = 0`, update `p_0` with PF PVR (eq. PF_pvr),
trivial evolution NOT used. `Δ_η` tuned per problem, decreased with resolution.

## 4. Reconstruction & derivatives (Sec. III D, App. C)

### WENO5 right-interface ENO stencils — `eq:WENO_right_stencils` (line 1720)
```
v^0_{i+1/2} = -1/6 p̄_{i-2} + 5/6 p̄_{i-1} + 1/3 p̄_i
v^1_{i+1/2} =  1/3 p̄_{i-1} + 5/6 p̄_i   - 1/6 p̄_{i+1}
v^2_{i+1/2} = 11/6 p̄_i    - 7/6 p̄_{i+1} + 1/3 p̄_{i+2}
```
Left-interface stencils `u^k_{i+1/2}` (line 1766) are the mirror images.

### Smoothness indicators (line 1732)
```
β^0 = 1/4 (3 p̄_i - 4 p̄_{i+1} + p̄_{i+2})² + 13/12 (p̄_i - 2 p̄_{i+1} + p̄_{i+2})²
β^1 = 1/4 (p̄_{i+1} - p̄_{i-1})²          + 13/12 (p̄_{i-1} - 2 p̄_i + p̄_{i+1})²
β^2 = 1/4 (p̄_{i-2} - 4 p̄_{i-1} + 3 p̄_i)² + 13/12 (p̄_{i-2} - 2 p̄_{i-1} + p̄_i)²
```

### Nonlinear weights — `eq:WENO_weights` (line 1742)  [exact constants]
```
w_k = α_k / Σ_l α_l,   α_k = d_k / (ε_W + β^k)²,   d_k = (3/10, 3/5, 1/10)
```
`d_k` give 5th-order accuracy in smooth regions. `ε_W` is a free regularizer:
`ε_W → ∞` ⇒ `w_k → d_k` (fixed 5th-order). Reconstructions:
`p^+_{i+1/2} = w_0 v^0 + w_1 v^1 + w_2 v^2`,
`p^-_{i+1/2} = w_0 u^0 + w_1 u^1 + w_2 u^2`.

### CWENO derivative — `eq:CWENO_derivative` (line 818)  [exact]
```
p̄'_i = [(p̄_{i-2} - 4 p̄_{i-1} + 3 p̄_i)/(2h)] w_0
      + [(p̄_{i+1} - p̄_{i-1})/(2h)] w_1
      + [(-3 p̄_i + 4 p̄_{i+1} - p̄_{i+2})/(2h)] w_2
```
Same `w_k` and `β^k` as WENO BUT **different linear weights** `d_k = (1/6, 2/3, 1/6)`
(line 826), giving **4th-order** accuracy in the derivative. Same `ε_W` used for
both reconstruction and derivative steps (empirically no benefit to splitting).

Curl constraint (`eq:curl_constraint`): `0 = ∂_x∂_y ξ − ∂_y∂_x ξ`; fixed stencils
satisfy exactly, CWENO satisfies only to `O(h²)`; recovered as `ε_W → ∞`.

## 5. Numerical flux — Kurganov–Tadmor — `eq:KT_flux` (line 895)
```
F_{i+1/2,j} = 1/2 ( f(p^-_{i+1/2,j}) + f(p^+_{i+1/2,j})
                   - a [ q(p^+_{i+1/2,j}) - q(p^-_{i+1/2,j}) ] )
```
**`a = 1` exactly** for BDNK (max propagation speed = 1 in this frame). KT with
`a = 1` ≡ HLL ≡ local Lax–Friedrichs. Same flux with `a = 1` used for `J^a` and
(for comparison) for relativistic Euler (whose true `|c_s| = 1/√3`). Flux is
symmetric in `p^±` ⇒ discrete conservation `∂_t ∫_D q dx dy = 0`
(`eq:discrete_cons`).

## 6. Time integration — Heun's method (TVD RK2) (line 966)
```
q̂^{n+1} = q^n + Δt H(q^n)
q^{n+1} = q^n + (Δt/2)[ H(q^n) + H(q̂^{n+1}) ]
```
Time step `Δt = λ h / a`, `a = 1`. Equal grid spacing `h` in x and y;
`h = (L_+ - L_-)/N_x`.

### Courant factors — Table I (`table:courant`, line 1039)  [the only data table]

|---|---|---|
| 1D Gaussian | 0.5 | 0.1 |

`λ ≡ aΔt/h`; `λ=0.5` is the CFL max for a 2D Cartesian grid. When the Pandya 2021

`v^x,v^y ≳ 0.98` at λ=0.1; stabilized by reducing to `λ = 0.05` or `0.01`.

### Boundary conditions
Outermost **3** grid cells per direction are boundary cells. Ghost cells
(`eq:ghost_cells`): `X_{k,j} := X_{3,j}` for `k∈[0,2]`,
`X_{k,j} := X_{N-4,j}` for `k∈[N-3,N-1]`. Periodic option also used (KH test).

## 9. Convergence tests (Appendix D)  [EXACT ORDERS]

Scheme is **second-order accurate** overall. Independent residual `R_N` is a
**second-order Crank–Nicolson** finite-difference discretization of the
x-component `∇_c T^{cx}=0` (independent from the solver discretization). For a
2nd-order scheme, `R_N ∝ O(h²)`. Convergence factor:
```
Q_N(t) = ||R_{N/2}|| / ||R_N||,   ||X||_1 ≡ Σ_i |X_i|  (1-norm)
```
**Expected smooth-limit value: `Q_N(t) → 4` as `h → 0`** (i.e. order-2 ⇒ factor 4).

Fig. `Conv_plot.pdf` (FIGURE-ONLY for the curves):
- Viscous rotor & KH columns: `Q_N(t)` approaches **4** as resolution increases;
  resolutions `N_x = 2^8, 2^9, 2^{10}`. Inviscid cases lose convergence at late
  times (grid-scale features); loss-time pushed later with resolution.
- 2D oblique shock column: residual `R_N` on constant-x slices at `t=220`, scaled
  so curves overlap at 2nd order: **`N_x=2^9` curve ×4, `N_x=2^{10}` curve ×16**.
  Smooth slices overlap (2nd order); shock slice shows delta-function spikes that
  grow taller/narrower with resolution (unresolved shock), 2nd order elsewhere.
- Self-convergence of `{ξ, n, u^x, u^y}` also confirmed 2nd order.

## 10. Reproduction recipe (minimal, for s1c.hrsc_core)

To reproduce the core HRSC method and its convergence:
1. Conformal BDNK in 4D Minkowski, evolve `q=(T^{tt},T^{tx},T^{ty})` and `J^t`.
   Primitives `p_0=(ε,u^x,u^y)` (evolve `ξ=ln ε`), `p_1=(ξ̇,u̇^x,u̇^y)`.
2. Transport: `η=η_0 ε^{3/4}`, `χ_0=(25/4)η_0`, `λ_0=(25/7)η_0` ⇒ char. speeds =1.
   Coordinate coefficients A, Q^x, Q^y, σ^{ij} from Appendix A above.
3. CWENO derivatives (`d_k=(1/6,2/3,1/6)`, 4th order) for `∂_i p_0`; WENO5
   reconstruction (`d_k=(3/10,3/5,1/10)`, 5th order) of `p_0` and `∂_i p_0` to
   faces; smoothness indicators β^k; single regularizer `ε_W` (tune per problem,
   `ε_W=1` for sharp 2D shocks).
4. KT flux with `a=1` (eq. KT_flux); same flux for `J^a`.
5. Adaptive PVR (shifted_BDNK_pvr + Δ_η switch to PF PVR) for small η_0.
6. Heun RK2 time step, `Δt=λh`, λ from Table I; 3 ghost/periodic boundary cells.
7. Convergence check: Crank–Nicolson 2nd-order independent residual of `∇_c T^{cx}`,
   1-norm `Q_N → 4`; for shocks, residual scales as ×4 (N→2N), ×16 (N→4N).

**Hard target for s1c (steady-state shock):** `ε_L=1, v_L=0.8` ⇒
`ε_R=119/27≈4.40741, v_R=5/12≈0.41667`; `η_0=0.2`, `w=10`, `L=200`, `λ=0.1`;
adaptive-PVR error → machine precision for `Δ_η ≲ 10^{-7}`.
