## 1. Equation of State (eq. label `EOS`, the "eq.53" in prompt numbering)

The combined polytropic + ideal-gas EoS, with Gamma=2 reducing to a quadratic, final closed form:

```
p(epsilon) = ( 1 + 2*epsilon*kappa - sqrt(1 + 4*epsilon*kappa) ) / (2*kappa)      (eq. EOS)
```

Built from:
- Polytropic (eq. `EOS:polytropic`): `p = kappa * rho_0^Gamma`
- Ideal gas (eq. `EOS:ideal_gas`): `p = (Gamma-1) * epsilon_0 * rho_0`
- `epsilon = rho_0 (1 + epsilon_0)` (eq. `epsilon_definition`)
- Inverted intermediate: `epsilon = (p/kappa)^(1/Gamma) + p/(Gamma-1)`

**EoS parameters (verbatim, Sec. EoS / line 436):**
- `kappa = 100`
- `Gamma = 2`
(positive root discarded by enforcing p(epsilon=0)=0)

**Central density convention (line 597):**
- Central rest-mass density: `rho_{0,c} = 0.00128 M_sun^-2`
- Equivalently central energy density: `epsilon_c = 0.00144 M_sun^-2`
- Total gravitational mass: `M_T = 1.4 M_sun`

NOTE: This is the ONLY TOV/stellar configuration in the paper. There is no TOV mass table; M_T = 1.4 M_sun is the single target value.

NOTE on prompt's "highB-F*": the paper has exactly FOUR cases — smallSB-F2, medS-F2, highB-F9, medSB-F9. There is no highB-F2 etc.; the frame label "F2" vs "F9" just denotes the two distinct frames (tau_eps = 0.023 vs 0.092).

## 2. Frame / transport parametrization (Sec. `frame`, eq. `hatted_parameters`)

Definitions:
- `rho := epsilon + p`
- `V := (4/3) eta + zeta`
- `c_s^2 := dp/depsilon` (fixed by EoS)
- `L` = characteristic length scale, **set to L = 1** (line 530)

Hatted parametrization (eq. `hatted_parameters`):
```
eta        = q_hat * L * c_s^2 * rho * eta_hat
zeta       = q_hat * L * c_s^2 * rho * zeta_hat
V_hat      = (4/3) eta_hat + zeta_hat
beta_eps   = c_s^2 * a_hat * V_hat * L
tau_p      = s_hat * c_s^2 * L * V_hat
tau_Q      = a_hat * L * V_hat
tau_eps    = V_hat * L
```
Additional frame choice (line 191): `beta_eps = tau_Q * p'(epsilon)`, giving
`Q^mu = tau_Q[(epsilon+p) u^nu nabla_nu u^mu + p'(epsilon) Delta^{mu nu} nabla_nu epsilon]`.

Positivity (eq. `hatted_parameters_positive`): `a_hat, q_hat, s_hat, L, c_s, eta_hat, V_hat > 0`; `zeta_hat >= 0`.

**Fixed frame parameters chosen for ALL simulations (Sec. "Choice of theory parameters", line 605-607):**
- `s_hat = 1`  (=> tau_p = c_s^2 * tau_eps)
- `q_hat = 0.999`  (just below upper bound q_hat < s_hat)
- `a_hat = 1`  (=> tau_Q = tau_eps)

With these, characteristic speeds (line 609):
```
c_0 = 0.9995 * sqrt(eta_hat / V_hat) * c_s
c_+ = 1.732 * c_s
c_- = 0.0183 * c_s
```

## 3. Hyperbolicity / causality / stability conditions

- Characteristic velocities flat space (eqs. `characteristic_velocities_1a`, `_1b`):
```
c_0   = c_s * sqrt( q_hat*eta_hat / (a_hat*V_hat) )
c_pm  = c_s * sqrt( [ a_hat(1+s_hat) + q_hat +/- sqrt( q_hat^2 + a_hat^2(4 q_hat + (s_hat-1)^2) + 2 a_hat q_hat (1+s_hat) ) ] / (2 a_hat) )
```
- Strong hyperbolicity + linear stability (eq. `cond_1`): `0 < q_hat < s_hat`
- Causality (eq. `cond_3`): `q_hat < ((1-c_s^2)/c_s^2) * ((1-s_hat c_s^2)/(c_s^2 + a_hat^{-1}))`
- Causality (eq. `cond_4`): `s_hat < 1/c_s^2`
- For the chosen params these hold as long as `c_s^2 < 1/3`.

**STABLE-WINDOW BOUNDARY (line 615):**
```
tau_eps = (4/3) eta_hat + zeta_hat  <~  0.1
```
(empirical bound for stable high-res evolution; found with Delta r = 0.001 M_sun)

c_s^2 bound verified in sims (line 653): max c_s approx 0.45 (c_s^2 approx 0.2 < 1/3) at the NS center, decreasing toward surface.

## 9. Numerical method / grid / CFL (Sec. `NM`)

- Method of Lines.
- Time integration: 3rd-order Strong Stability Preserving Runge-Kutta (SSP-RK3).
- **CFL / Courant factor: `Delta t / Delta r = 0.25`.**
- Spatial: 3rd-order finite-volume = 4th-order finite difference with 3rd-order dissipation ("finite-difference Osher-Chakrabarthy", FDOC).
- Dissipation uses max of the 6 characteristic velocities (eq. `vel_w_m`); **minimum velocity floor = 0.1 c**.
- Atmosphere: identified where `p < kappa rho_{0,atms}^Gamma`, with `rho_{0,atms} = 1e-12 M_sun^-2`; atmosphere points set to `rho_0 = 1e-13 M_sun^-2`, velocity and time-derivatives (eps_hat, vbar_hat^r) set to zero.
- **Resolutions: Delta r = [0.001 - 0.0032] M_sun.** QNM/decay table set: [0.002, 0.0024, 0.0028, 0.0032]. Convergence-test set: [0.001, 0.002, 0.0028].
- **Box size: r_max = 20 M_sun.** Staggered grid (avoids r=0). Outflow BC for fluid at outer boundary.
- Evolution time: t_f = 8000 M_sun (except Delta r=0.001 reached t_f=4500 M_sun due to cost). PF decay-rate runs extended to t_f = 20000 M_sun.
- epsilon_c extracted every Delta t = 1 M_sun.

Evolved variables (spherical): `{ gamma_tilde*E, gamma_tilde*S_r, epsilon, d_r epsilon, v_tilde^r, d_r v_tilde^r }`, with `gamma_tilde = sqrt(g_rr) g_theta_theta`, `v_tilde^r = v^r / r`. Primitives p0=(epsilon, v_tilde^r), p1=(eps_hat, vbar_hat^r); conserved q=(gamma_tilde E, gamma_tilde S_r). contoprim is a LINEAR 2x2 (spherical) / 4x4 (3D) solve (Appendix `con2prim_appendix`).

## 10. Decay-rate extraction method (Sec. `QNM_decay_rate_section`)

- Butterworth filter order 4 applied to raw epsilon_c -> clean signal eps_tilde_c.
- Frequency cutoff window `[0.01, f_sampling/10]` (code units 1/M_sun).
- Linear method: log|eps_tilde_c|, extract maxima, straight-line fit -> slope = decay rate.
- Nonlinear method (fit function): `eps_tilde_c(t) = A exp(-t/tau) cos(omega t + phi_0) + C`.
- Extrapolation averaged over fitting methods/windows, kept to 2 sig figs.
