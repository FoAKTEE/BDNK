## 1. Conventions / units (Sec. Model, Table `table:notation`)

- Metric signature mostly-plus `(-+++)`; geometric units `c = 1`.
- Energy density `\epsilon`; specific internal energy `e`; pressure `P`; `\rho \equiv \epsilon + P`; rest-mass density `m n`; adiabatic index `\Gamma`.
- `L = 1` set throughout (the dimensionful frame lengthscale; eq:hydro_frame). **This is a hard convention to match.**
- All viscous test problems are flat-space, variation in at most one spatial dimension; bulk + shear viscosities only ever appear in the combination `V` (eq:V).

---

## 2. Equation of state — relativistic ideal gas "gamma-law"

**EOS (eq:EOS):**
```
P(eps,n) = (Gamma-1) m n e(eps,n) = n T(eps,n)
```
with `1 < Gamma < 2`.

**Internal-energy relation (eq:e_defn):** `eps = m n (1 + e)`, i.e. `e = eps/(m n) - 1`.

**Entropy density (eq:S_over_V):**
`s(eps,n) = m n [ (1/((Gamma-1) m)) ln( e / n^{Gamma-1} ) + const ]`

**Relativistic chemical potential (eq:mu):**
`mu = m + m e ( Gamma - ln( e / n^{Gamma-1} ) + const )`

**Sound speed squared (eq:cs_sq):** `c_s^2 = Gamma P / rho`.

## 3. BDNK constitutive relations (the "ideal-gas constitutive eqs", task's "25-37")

Currents are decomposed (eq:Tab, eq:Ja) and closed by the BDNK definitions:

| label | equation |
|---|---|
| eq:script_E | `E = eps + tau_eps [ u^c grad_c eps + rho grad_c u^c ]` |
| eq:script_P | `P_scr = P - zeta grad_c u^c + tau_P [ u^c grad_c eps + rho grad_c u^c ]` |
| eq:Q_a | `Q^a = tau_Q rho u^c grad_c u^a + beta_eps Delta^{ac} grad_c eps + beta_n Delta^{ac} grad_c n` |
| eq:script_T_ab | `T^{ab} = -2 eta sigma^{ab} = -2 eta grad^{<a} u^{b>}` |
| eq:script_N | `N = n` |
| eq:script_J_a | `J^a = 0` |
| eq:rho | `rho = eps + P` |

**beta coefficients (eq:beta_eps, eq:beta_n):**
```
beta_eps = tau_Q p'_eps + (sigma/rho) kappa_eps
beta_n   = tau_Q p'_n   + (sigma/n)   kappa_n
```

**Microphysics derivatives, evaluated for the ideal gas (eqs after kappa_n_defn):**
```
p'_eps  = Gamma - 1
p'_n    = -(Gamma - 1) m
kappa_eps = -(Gamma-1) eps rho^2 / (n^2 P)
kappa_n   = (rho/(n^2 P)) [ (Gamma-1) eps^2 + P^2 ]
kappa_s = kappa_eps + kappa_n = -(Gamma-1) m rho / n
```
so for the relativistic ideal gas:
```
beta_eps = (Gamma-1) tau_Q - (Gamma-1) sigma eps rho / (n^2 P)
beta_n   = -(Gamma-1) m tau_Q + (sigma rho/(n^3 P)) [ (Gamma-1) eps^2 + P^2 ]
```

**Auxiliary dimensionless quantities (eq:omega, eq:alpha):**
```
omega = kappa_s/kappa_eps = m n P/(eps rho)
alpha = p'_eps/c_s^2 = (Gamma-1)/c_s^2
```

Conservation laws to evolve (eq:Tab_cons_law, eq:Ja_cons_law): `grad_a T^{ab}=0`, `grad_a J^a=0`. Five EOM for `eps, n, u^i`.

## 4. Hydrodynamic frame choice (eq:hydro_frame, eq:V, eq:Vhat_defn)

```
eta   = rho c_s^2 L  eta_hat
zeta  = rho c_s^2 L  zeta_hat
sigma = (Vhat L rho c_s^2 / (-kappa_eps)) sigma_hat
tau_eps = tau_Q = L Vhat tau_hat
tau_P   = 2 (Gamma-1) L Vhat        [main text] ; equivalently tau_P = 2 alpha c_s^2 L Vhat  [eq:frame_ansatz; equal since alpha c_s^2 = Gamma-1]
V    = 4 eta/3 + zeta
Vhat = V/(rho c_s^2 L) = Re^{-1}
L = 1
```
Free dimensionless params: `eta_hat>0, zeta_hat>=0, sigma_hat>=0, tau_hat` (sets max char speed `c_+`).

**Frame constraints (eq:simple_constraints) — the simple causal/stability bounds:**
```
sigma_hat <= 1/3                              (linear stability)
tau_hat   >= [ (Gamma-1)(2 - c_s^2) + c_s^2 ] / (1 - c_s^2)   (causality)
```
Even simpler causal bound (footnote, Gamma->2): `tau_hat >= 2/(1-c_s^2)`.

### 6a. Bjorken flow (Sec. Bjorken_flow) — ODE + 4th-order convergence Q_N=16

**EOM (eq:Bjorken_EOM), Milne coords, metric diag(-1,1,1,tau^2):**
```
tau_eps eps'' = -(1/tau)(tau + 2 tau_eps + tau_P) eps'
                - (1/tau^2)[ rho (tau + tau_P) - V ]
```
(`'` = d/dtau). Only the tau-component of grad_a T^{ab}=0 is nontrivial. `Q^a=0`, sigma & tau_Q absent. `n(tau)=n_0/tau`.

**Inviscid reference solution (eq:inviscid_bjorken):**
`eps(tau) = m n_0 tau^{-1} [ 1 + e_0 tau^{-(Gamma-1)} ]`.
Diagnostic quantity plotted: `eps' + Gamma eps/tau`, which equals `m n_0 (Gamma-1)/tau^2` on the inviscid solution (independent of `e_0`).

**EXACT initial conditions (text + fig:bjorken caption):**
- `n_0 = 0.1`
- `eps_0 = eps(tau=1) = 0.25`
- `eps_dot_0 in {-2, 0, 2}`
- integrate `tau = 1 -> tau = 20`
- `Gamma=4/3, m=1, Vhat=1/10, sigma_hat=0, tau_hat in {0.5,1,2}` (Table)

**Char speeds reported in text/caption (figure read-offs, see note):**
- `tau_hat=0.5`: always superluminal, `c_+ ≈ 1.3`
- `tau_hat=1`: superluminal early (`c_+ ≈ 1.05`), subluminal late (`c_+ ≈ 0.9`)
- `tau_hat=2`: always subluminal, `c_+ ≈ 0.7`
> NOTE: these `c_+` values are quoted approximately in text; they are not a precise table. Best read as ~2-sig-fig descriptive values.

**Convergence target (Table `table:ODE_conv`, eq:convergence_factor `Q_N = ||R_{N/2}||/||R_N||`):**
RK4 ODE integration. Residual `R_N` = independent 4th-order centered FD discretization of eq:Bjorken_EOM. Results for `eps_dot=-2` (others "essentially identical"):

### 6b. 1D steady planar shockwave (Sec. shockwaves) — exact ICs + WENO5 PDE convergence Q_N=4

**Steady-state ODE system (shock rest frame, u^a=(W, Wv,0,0)):**
- eq:shockwave_nprime: `n' = - W^2 n v'/v`
- eq:shockwave_epsP: `eps'(x) = (c4 v^4 + c3 v^3 + c2 v^2 + c1 v + c0)/(A W v (v-c_+)(v+c_+)(v-c_-)(v+c_-))`
- eq:shockwave_velP: `v'(x) = (d3 v^3 + d2 v^2 + d1 v + d0)/(A W^3 (v-c_+)(v+c_+)(v-c_-)(v+c_-))`

Coefficients `c_i,d_i` (eq after shockwave_velP), with `A=rho tau_eps tau_Q` (eq:A), `B` (eq:B), `C` (eq:C); `delta = beta_eps rho + beta_n n - rho c_s^2 tau_Q - sigma kappa_s = 0` identically.
Char speeds `c_pm^2 = (-B ± sqrt(B^2-4AC))/(2A)` (eq:cpmsq_general).

`T^{tx}, T^{xx}` constants computed from perfect-fluid stress-energy of the LEFT state:
`T^{tx} = rho W^2 v`, `T^{xx} = rho W^2 v^2 + P` (eq:Tab_0).

**Steady-state shock PROFILE (fig:shockwave_profile):** asymptotic left state
`{eps_L, v_L, n_L} = {1, 0.8, 0.1}` as x->-inf; `Gamma=4/3, m=0.1, Vhat=2/15, sigma_hat=0, tau_hat=1.5`.

## 7. Frame coefficient choices for comparison theories

**Eckart frame (eq:Eckart_frame):** `tau_eps = tau_P = 0`, `tau_Q = -kappa T/rho`, with `kappa = sigma rho^2/(n^2 T)` (eq:kappa). `J^a = n u^a`.

**Trivial-equilibrium relaxation comparison (eq:theory_comp), constant tau:**
```
Eckart: eps = T^{tt}
BDNK:   eps_dot = (1/tau_eps)(T^{tt} - eps)
MIS:    eps_dot = (1/(tau_pi + c_1))(T^{tt} - eps)
```
Identification `tau_eps = tau_pi + c_1 = tau`. Exponential solution (eq:exp_relax):
`eps(t) = T^{tt} + (eps_0 - T^{tt}) e^{-t/tau}`.

**Bjorken positivity bound (eq:Pmin, eq:tau_phys):** `P > ((Gamma-1)/Gamma)(eta/tau_Q - m n)`; optional `tau_Q > eta/(m n)` (NOT enforced in paper). The required causality/stability constraint is `rho > eta/tau_Q`.

## 8. Heat-flow (Sec. heat_flow) — telegrapher's-equation tests

**Alt heat vector (eq:alt_heat_vector):** `Q^a = -kappa Delta^{ab} grad_b T + tau_Q rho u^b grad_b u^a + gamma Delta^{ab} grad_b P`, `gamma = tau_Q + sigma rho/n^2` (eq:gamma).

**EOM (eq:heat_t_eqn, eq:heat_x_eqn):**
```
0 = (eps + tau_eps eps_dot)_{,t} + (-kappa T' + gamma P')_{,x}
0 = (-kappa T' + gamma P')_{,t} + (P + tau_P eps_dot)_{,x}
```

**Constant-coefficient reduced equations:**
- Eckart (eq:heat_t_Eckart): `T_dot - alpha_E T'' = 0`, `alpha_E = kappa(Gamma-1)/n`.
- Hybrid (eq:heat_t_hybrid): `T_ddot - c_h^2 T'' + (1/tau_eps) T_dot = 0`, `c_h^2 = kappa(Gamma-1)/(n tau_eps)`.
- BDNK (eq:heat_t_BDNK): `T_ddot - c_B^2 T'' + (1/tau_eps) T_dot + l.o.t. = 0`, `c_B^2 = c_h^2 (1 - gamma n/kappa)`, `l.o.t. = ((Gamma-1)/(n tau_eps)) gamma (n'' T + 2 n' T')`.

Pressure relaxation (eq:heat_x_soln): `P_dot = (1/tau_theta)(P_0(t)-P)`, `tau_theta = theta/n`, with `theta` per eq:heat_theta_defn (Eckart/hybrid: `-kappa`; BDNK: `-kappa + gamma n + tau_P n/(Gamma-1)`).

**Heat-flow ICs (eq:heat_flow_ID), time-symmetric:**
```
T(0,x) = A e^{-x^2/w^2} + delta,   P(0,x) = P_0 = const
eps = P[m T^{-1} + (Gamma-1)^{-1}],  n = P T^{-1}
eps_dot(0,x) = u_dot^i(0,x) = 0
```
> NOTE: numeric values of `A, w, delta, P_0` are NOT given explicitly in the text; only the functional form. They must be inferred from figs (fig:heat_stationary, fig:telegraphers).

**fig:heat_stationary:** sigma_hat in {0, 1/3}, tau_hat=1.5, Vhat=2/15. Target: eps_dot -> 0 (converges to zero) for sigma_hat=0; eps_dot -> finite nonzero for sigma_hat=1/3 (eq:heat_ID_EOM `tau_eps eps_ddot = (kappa T')'`).

**fig:telegraphers:** three runs with `sigma_hat/tau_hat = 0.1` fixed: `(sigma_hat,tau_hat) = (0.15,1.5),(1.5,15),(7.5,75)`, Vhat=2/15. sigma_hat=7.5 case shows central peak splitting (wave behavior) then oscillatory instability that crashes (violates sigma_hat<=1/3). Reference: cf. Most:2021rhr Fig. 2.

## 9. Causality / characteristic speeds (Appendix hydro_frame) — task's "eq44/71"

**Characteristic speeds for frame eq:hydro_frame (eq:cpmsq, the squared max/min speeds):**
```
c_pm^2 = (c_s^2/(2 tau_hat)) ( 2 alpha - omega sigma_hat + tau_hat + 1
         ± [ omega sigma_hat (4 alpha + omega sigma_hat) + (2 alpha+1)^2
             - 2(omega+2) sigma_hat + tau_hat^2 + tau_hat(2 - 2 omega sigma_hat) ]^{1/2} )
```
**(eq:c1sq):** `c_1^2 = c_s^2 eta/(V tau_hat)`.

**Causality constraints (eq:CAUS_A..D = BDNK eq. (20)):**
```
rho tau_Q > eta                  (CAUS A)
B^2 >= 4 A C >= 0                 (CAUS B)
2A > -B >= 0                      (CAUS C)
A > -B - C                        (CAUS D)
```
shorthand: A=rho tau_eps tau_Q; B=-tau_eps(rho c_s^2 tau_Q + V + sigma kappa_s) - rho tau_P tau_Q; C=tau_P(rho c_s^2 tau_Q + sigma kappa_s) - beta_eps V; D=rho c_s^2(tau_eps+tau_Q)+V+sigma kappa_eps; E=sigma(p'_eps kappa_s - c_s^2 kappa_eps).

**Linear stability (eq:STAB_A1..E = BDNK eq. (48))** — full set reproduced in source lines 1339-1357.

**Simplified causality constraints for frame ansatz (eq:caus_const_simplified):**
```
tau_hat > c_s^2 eta/(4eta/3 + zeta)
2 tau_hat > c_s^2 (2 alpha - omega sigma_hat + tau_hat + 1)
c_s^4(-2 alpha omega sigma_hat + sigma_hat + alpha tau_hat) + tau_hat^2
   >= c_s^2 tau_hat (2 alpha - omega sigma_hat + tau_hat + 1)
```

**Fully-simplified single causal bound (eq:fully_simplified_caus_const) — TASK'S CAUSALITY BOUND:**
```
tau_hat >= [ (Gamma-1)(2 - c_s^2) + c_s^2 ] / (1 - c_s^2)
```
(same as eq:simple_constraints; saturating gives |c_+|<1 for sigma_hat>0, |c_+|=1 in sigma->0 limit). Use `>` instead of `>=` to enforce strictly subluminal |c_+|<1.

**Second-law / parameter ranges (eq:trans_coeff_ranges):**
```
rho,n,tau_eps,tau_P,tau_Q,eta > 0;  m,zeta,sigma >= 0
0 < c_s^2 < 1;  0 < omega < 3 - 2 sqrt(2) ≈ 0.2;  1 <= alpha
```
Simple stability constraint reduces to `sigma_hat <= 1/2` (eq:simple_stab_const: `1-(2-alpha omega)sigma_hat>=0`), tightened to `sigma_hat <= 1/3` (eq:sigma_bound) for the full set.

**Frame ansatz used in appendix (eq:frame_ansatz):** `tau_eps = tau_Q = L Vhat tau_hat`, `tau_P = 2 alpha c_s^2 L Vhat`.

## 10. Numerical method summary (Appendix Numerics)

- **ODEs (Bjorken, steady shock):** RK4 (4th-order explicit Runge-Kutta). Resolutions N = 2^9 .. 2^13. Convergence `Q_N -> 16`.
- **PDEs (dynamical shock, heat flow):** conservative finite-volume of Pandya:2022pif; method of lines + TVD RK2 (Heun's method); WENO/CWENO spatial reconstruction (<=4th order for smooth flows); overall 2nd-order. CFL `lambda = 0.1` (0.01 for stiff cases). Convergence `Q_N -> 4`.
- `Q_N = ||R_{N/2}||/||R_N||` (1-norm), residual from an INDEPENDENT discretization (4th-order centered FD for ODE residuals; 2nd-order Crank-Nicolson for PDE residual of `grad_a T^{ab}=0` t-component).
