## 1. THE FRAME / CONSTITUTIVE RELATIONS (the "BDNK-Eckart" frame used here)

This work uses an **Eckart-like frame**: the flow is tied to the baryon current, `J^μ = n u^μ` throughout (so γ_i = ν_i = 0). Dynamical variables: **ε (energy density), n (baryon density), u^μ**.

Constitutive relations [Eq. (finaltheory), lines 1048–1057]:
```
J^μ  = n u^μ
T^μν = (ε + A) u^μ u^ν + (P + Π) Δ^μν − 2η σ^μν + u^μ Q^ν + u^ν Q^μ
A    = τ_ε [ u^λ ∇_λ ε + (ε+P) ∇_λ u^λ ]                                    (Eq. 10c)
Π    = −ζ ∇_λ u^λ + τ_P [ u^λ ∇_λ ε + (ε+P) ∇_λ u^λ ]
Q^ν  = τ_Q (ε+P) u^λ ∇_λ u^ν + β_ε Δ^{νλ}∇_λ ε + β_n Δ^{νλ}∇_λ n
```
with [Eq. (definebetas), lines 1059–1065]:
```
β_ε = τ_Q (∂P/∂ε)_n + [σ T (ε+P)/n] (∂(μ/T)/∂ε)_n
β_n = τ_Q (∂P/∂n)_ε + [σ T (ε+P)/n] (∂(μ/T)/∂n)_ε
```
- σ^μν = Δ^{μναβ}∇_α u_β is the shear tensor.
- ρ ≡ ε + P (used throughout).
- Transport parameters of the model: **η (shear visc.), ζ (bulk visc.), σ (charge/heat conductivity), and three relaxation times τ_ε, τ_P, τ_Q** (each with dimensions of time). On-shell A ~ O(∂²), Π ~ −ζ∇·u + O(∂²), Q_ν ~ σT(ε+P)/n Δ∇(μ/T) + O(∂²), i.e. reduces to Eckart at first order.

**Units / conventions:** Metric signature (−,+,+,+), u_μ u^μ = −1, harmonic gauge g^{μν}Γ^α_{μν}=0 for the causality proof; geometrized factor kept explicit (8πG in Einstein eqs, line 958). Quantities are in natural units (no specific physical unit system fixed — all constraints are dimensionally homogeneous and frame-independent under field redefinitions for η, ζ, σ).

Speed of sound [Eq. soundspeed, line 1153]:

```

p'_ε = (∂P/∂ε)_n ,  p'_n = (∂P/∂n)_ε

```

## 2. TARGET A — CAUSALITY (Theorem I) — THE DEFINITIVE INEQUALITIES

**Theorem I** (lines 1164–1188). Assumption (A1) [line 1171]:
```
(A1):  ρ = ε+P,  τ_ε, τ_Q, τ_P > 0   and   η, ζ, σ ≥ 0.
```
Under (A1), causality holds **if and only if** the following hold [Eq. (C_conditions), lines 1176–1186]:

```
(C-a)  ρ τ_Q > η                                                           (Eq. C_condition_a)

(C-b)  [ τ_ε( ρ c_s² τ_Q + ζ + 4η/3 + σ κ_s ) + ρ τ_P τ_Q ]²
         ≥ 4 ρ τ_ε τ_Q [ τ_P( ρ c_s² τ_Q + σ κ_s ) − β_ε( ζ + 4η/3 ) ] ≥ 0
                                                                           (Eq. C_condition_b)

(C-d)  2 ρ τ_ε τ_Q > τ_ε( ρ c_s² τ_Q + ζ + 4η/3 + σ κ_s ) + ρ τ_P τ_Q ≥ 0
                                                                           (Eq. C_condition_d)

(C-e)  ρ τ_ε τ_Q + σ κ_s τ_P
         > τ_ε( ρ c_s² τ_Q + ζ + 4η/3 + σ κ_s ) + ρ τ_P τ_Q (1 − c_s²) + β_ε( ζ + 4η/3 )
                                                                           (Eq. C_condition_e)
```
"The same result holds true for [the fluid] equations if the metric is not dynamical." (line 1187) — i.e. flat-space BDNK obeys the same causality conditions.

### Characteristic structure underlying Theorem I (Appendix, "Proof of Theorem I", lines 1763–1824)
The matter-sector characteristic determinant factorizes [Eq. (det_a/det_b), lines 1803–1804]:
```
det[M(ξ)] = −b² [ ρ τ_Q b² − η(v·v) ]³ [ A b⁴ + B b²(v·v) + C(v·v)² ]
          = −ρ⁴ τ_Q⁴ τ_ε (u·ξ)² ∏_{a=1,±} [ (u·ξ)² − c_a Δ^{αβ}ξ_α ξ_β ]^{n_a}
```
with b ≡ u^α ξ_α, v^α ≡ Δ^{αβ}ξ_β, v·v = Δ^{αβ}ξ_α ξ_β, and [Eq. (ABC), lines 1808–1815]:
```
A ≡ ρ τ_ε τ_Q
B ≡ −τ_ε( ρ c_s² τ_Q + ζ + 4η/3 + σ κ_s ) − ρ τ_P τ_Q
C ≡ τ_P( ρ c_s² τ_Q + σ κ_s ) − β_ε( ζ + 4η/3 )
```
Multiplicities/characteristic speeds: n_1 = 3, n_± = 1;
```
c_1 = η/(ρ τ_Q)          [printed in source as η/(ρ τ_s); τ_s is a typo for τ_Q — consistent with (C-a)]
c_± = [ −B ± sqrt(B² − 4AC) ] / (2A)
```
Causality (C1: roots real; C2: ξ_μ ξ^μ ≥ 0) ⇒ each characteristic-speed-squared c_a must satisfy **0 ≤ c_a < 1** (strict < 1 chosen so matter signals stay strictly sub-luminal; footnote, lines 1816–1822). Mapping:
- c_1 causal ⇔ (C-a) `ρτ_Q > η`.
- c_± real ⇔ first inequality of (C-b), `B²−4AC ≥ 0`.
- c_± ≥ 0 ⇔ `−B ≥ 0` [2nd ineq. of (C-d)] and `C ≥ 0` [2nd ineq. of (C-b)].
- c_± < 1 ⇔ `2A + B > 0` [1st ineq. of (C-d)] and `A + B + C > 0` [(C-e)].
Identity used: `β_ε + n β_n/ρ = τ_Q c_s² + σ κ_s/ρ` (line 1816).

## 3. TARGET B — STRONG HYPERBOLICITY & WELL-POSEDNESS (Proposition I, Theorem II)

**Proposition I** (lines 1340–1356): under (A1) with **η > 0** AND (C_conditions) holding in **strict** form (`>` instead of `≥`), for any timelike co-vector ξ: (i) det(𝔄^α ξ_α) ≠ 0; (ii) for any spacelike ζ, the eigenvalue problem (ζ_α + Λ ξ_α)𝔄^α R = 0 has only **real eigenvalues Λ** and a **complete set of right eigenvectors** ⇒ strong hyperbolicity.

**Theorem II** (lines 1393–1411): local well-posedness (existence + uniqueness of a maximal globally hyperbolic development) for the Einstein-fluid system, given (A1) with η>0, (C_conditions) strict, analytic transport coefficients, n̊ > 0, and Sobolev data g̊,ε̊,n̊,ů ∈ H^N, with N ≥ 5 (and the first time-derivatives in H^{N−1}).

## 4. TARGET C — LINEAR STABILITY (boost theorem + explicit conditions)

**Theorem III** (lines 1494–1497): If the linearized system (s_2-1) has N linearly independent real eigenvectors, and the system is **causal AND stable in the local rest frame (LRF)**, then it is **stable in any Lorentz-boosted frame**. (This is the headline general result: rest-frame stability + causality + diagonalizability ⇒ stability in all frames.)

**Conditions for linear stability** (Cowling approx., g_μν = η_μν = diag(−1,1,1,1), δg_μν = 0). The system (EOMcurrent) is linearly stable if it is causal in the **strict** form of (C_conditions), with η > 0, AND [Eq. (stability), lines 1599–1611]:
```
(S-a)  (τ_ε+τ_Q)|B| ≥ τ_ε τ_Q D ≥ ρ c_s² τ_ε τ_Q (τ_ε+τ_Q)                  (stability_a)

(S-b)  (τ_ε+τ_Q)|B| D + ρ τ_ε τ_Q (τ_ε+τ_Q) E > τ_ε τ_Q D² + ρ(τ_ε+τ_Q)² C  (stability_b)

(S-c)  c_s² D − E ≥ ρ c_s⁴ (τ_ε+τ_Q)                                         (stability_c)

(S-d)  (τ_ε+τ_Q)[ |B|(c_s² D − 2E) + 2 c_s² ρ τ_ε τ_Q E + C D ]
         > 2 c_s² ρ(τ_ε+τ_Q)² C + τ_ε τ_Q D (c_s² D − E)                     (stability_d)

(S-e)  |B| D [ C(τ_ε+τ_Q) + E τ_ε τ_Q ] + 2 ρ τ_ε τ_Q (τ_ε+τ_Q) C E
         > ρ C²(τ_ε+τ_Q)² + τ_ε τ_Q ( C D² + ρ τ_ε τ_Q E² ) + B² E (τ_ε+τ_Q) (stability_e)
```
where B, C are as in (ABC) above, D, E as defined in Sec.1, and **|B| = −B > 0** (from strict (C-d)).

### Dispersion relations behind the stability conditions (LRF), [Eq. (modes), lines 1623–1640]
Dimensionless: τ̄_Q=Tτ_Q, τ̄_ε=Tτ_ε, η̄=Tη/ρ, B̄=(T²/ρ)B, C̄=(T²/ρ)C, D̄=(T/ρ)D, Ē=(T/ρ)E.
```
Shear channel:  τ̄_Q Γ² + η̄ k² + Γ = 0
Sound channel:  a_0 Γ⁵ + a_1 Γ⁴ + a_2 Γ³ + a_3 Γ² + a_4 Γ + a_5 = 0
  a_0 = τ̄_ε τ̄_Q
  a_1 = τ̄_ε + τ̄_Q
  a_2 = 1 + k²|B̄|
  a_3 = k² D̄
  a_4 = c_s² k² + k⁴ C̄
  a_5 = k⁴ Ē
```
Stability requires Re(Γ) ≤ 0. Sound channel handled by **Routh-Hurwitz** [line 1656]: (i) a_0,a_1>0; (ii) a_1a_2−a_0a_3>0; (iii) a_3(a_1a_2−a_0a_3)−a_1(a_1a_4−a_0a_5)>0; (iv) (a_1a_4−a_0a_5)[...]−a_5(a_1a_2−a_0a_3)²>0; (v) a_5>0. Mapping: (ii)→1st ineq. (S-a); (iii)→2nd ineq. (S-a)+(S-b); (iv)→(S-c)–(S-e).
Shear channel: two roots with Re(Γ)≤0 iff τ_Q>0 and η≥0 [matches (A1)].

### Hydrodynamic-mode content (physical check, line 1658)
- Shear: diffusive hydro mode ω(k) = −i k² η/(ε+P) + … ; non-hydro mode ω = −i/τ_Q + …
- Sound: ω(k) = ±c_s k − i k² Γ_s/2 + … (Γ_s = Γ_s(η,ζ,σ) as in Eckart); heat-diffusion mode ω = −i D k² + … (D ~ σ); two non-hydro modes ω = −i/τ_ε and ω = −i/τ_Q at low k.

### Entropy production (line 1099–1102)
```
∇_μ S^μ = (2η σ_μν σ^μν)/T + ζ(∇_μ u^μ)²/T + σT[Δ∇(μ/T)][Δ∇(μ/T)] + O(∂³)  ≥ 0  iff  η,ζ,σ ≥ 0
```

## 6. RELEVANT TYPOS / GOTCHAS FOUND IN SOURCE
- Line 1816: `c_1 = η/(ρ τ_s)` — `τ_s` is a typo; should read `η/(ρ τ_Q)` (consistent with (C-a) and the determinant factor `ρτ_Q b² − η(v·v)`).
- D in the stability section uses **κ_ε** (not κ_s): `D = ρ c_s²(τ_ε+τ_Q) + ζ + 4η/3 + σκ_ε`. Do NOT confuse with the κ_s appearing in B, C and the causality conditions.
- The (C-b)/(C-d)/(C-e) conditions are written using the **κ_s** combination; B and C reuse those same groupings (B = −[ τ_ε(ρc_s²τ_Q+ζ+4η/3+σκ_s) + ρτ_Pτ_Q ], C = τ_P(ρc_s²τ_Q+σκ_s) − β_ε(ζ+4η/3)), and A = ρτ_ετ_Q.
- "Cowling approximation" (δg_μν=0, flat background) is used for the linear-stability derivation; the causality conditions are gauge-independent and hold in full GR.
