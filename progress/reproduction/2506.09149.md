## 1. EOS and Star Parameters

### 1.1 General EOS framework (Sec. II, III.A)
- EOS written as `[p(n,s), ε(n,s)]`, `s` = entropy per baryon, `n` = baryon number density.
- First law: `dε = (ε+p)/n dn + n T ds`.
- **Background star is isentropic** (`∇_μ s = 0`) in static spherical symmetry → EOS reduces to effectively 1D `p = p(n, s_0)`, `ε = ε(n, s_0)`, i.e. `p = p(ε)` (lines 511–512).
- Star radius `R` defined by `p(R) = 0`; they assume EOS such that this also implies `ε(R) = 0` (line 513). Exterior = Schwarzschild.
- **No specific neutron-star EOS table, no specific central density, no specific (M, R) values are given anywhere.**

### 1.2 The only concrete EOS instance: Mathews relativistic ideal gas (lines 1077–1090)
Used purely as an *illustrative example* to evaluate `c_n^2` and `c_s^2 - c_n^2`. NOT integrated into a star; no numbers produced.

EOS (Mathews 1971), particle mass `m`:
```
p_Mathews(ε, n) = (1/3) [ ε^2 - (m n)^2 ] / ε                       (no eq label)
```
Derived quantities:
```
c_n^2 := (∂p/∂ε)_n = (1/3) (m n / ε)^2  ≥ 0
c_s^2 - c_n^2 = [ n/(ε+p) ] (∂p/∂n)_ε = -(2/3) [ n/(ε+p) ] (m n / ε) m  ≤ 0
```
Consequence stated: `L ≥ 0` but `K ≤ 0` for the Mathews ideal gas → the sufficient stability condition (`L≥0 AND K≥0`) is **not** satisfied, while the necessary condition is automatically satisfied. No numeric evaluation given.

### 1.3 Speeds of sound (definitions; load-bearing)
```
γ ≡ ∂log p/∂log n |_s   (adiabatic index)
γ p = (ε+p) c_s^2,    c_s^2 ≡ (∂p/∂ε)_s        (lines 439-441)
c_n^2 ≡ (∂p/∂ε)_n                               (line 445)
```
Note (line 447): `c_n^2` is NOT necessarily positive nor `< 1` (unlike `c_s^2`). Causality+stability of the perfect fluid imposes `c_s^2 ∈ [0,1]`.

## 2. Transport / Frame Parameters (exact)

BDNK constitutive relations (Eq. set `eq:bdnk`, lines 297–306):
```
E_BDNK   = τ_E [ u^μ∇_μ ε + (ε+p)Θ ]                         (eq:bdnke)
P_BDNK   = -ζ Θ + τ_P [ u^μ∇_μ ε + (ε+p)Θ ]                  (eq:bdnkp)
Q^μ_BDNK = κ T (ε+p)/n  Π^{μλ}∇_λ φ  +  τ_Q [ (ε+p)a^μ + Π^{μλ}∇_λ p ]   (eq:bdnkq)
π^{μν}_BDNK = -2 η σ^{μν}
N_BDNK = 0,   J^μ_BDNK = 0
```
Fugacity: `φ ≡ (ε+p)/(nT) - s` (line 309).
Transport coefficients: `ζ` (bulk), `η` (shear), `κ` (heat conductivity), relaxation times `τ_E, τ_P, τ_Q`. All assumed functions of background `(n,s)` → functions of `r` only (lines 611).

Eckart (Eq. `eq:Eckart`, lines 315–322): `P_Eck=-ζΘ`, `Q^μ_Eck = κT(ε+p)/n Π^{μλ}∇_λφ`, `π=-2ησ`, `E_Eck=N_Eck=J_Eck=0`. **Recovered from BDNK by `τ_E, τ_P, τ_Q → 0`.**

MIS / Maxwell-Cattaneo (Eq. `eq:mc`, lines 355–362) with relaxation times `τ_0, τ_1, τ_2`:
```
τ_0 u^ν∇_ν P_MC + P_MC = -ζ Θ
τ_1 Π^μ_ν u^λ∇_λ Q^ν_MC + Q^μ_MC = κ T (ε+p)/n Π^{μλ}∇_λ φ          (eq:mcq)
τ_2 Π^μ_α Π^ν_β u^λ∇_λ π^{αβ}_MC + π^{μν}_MC = -2η σ^{μν}
E_MC = N_MC = J_MC = 0
```

**Small-viscosity bookkeeping parameter** `α ≪ 1`: rescale `(ζ,η,κ) → (αζ, αη, ακ)` (line 758). Relaxation times scale as `τ ~ α` (sets the fast timescale `τ = t/α`). All first-order stability results are at `O(α)`.

## 3. EXACT TARGETS (criteria + operators + closed forms to reproduce)

These are the verbatim equations a reproduction must match. No tolerances apply (analytic).

### 3.1 Chandrasekhar / Sturm–Liouville background operator (Eqs. `eq:chandra`, `eq:wpq`)
```
W(r) ∂_t Ẋ = ∂_r( P(r) ∂_r Ξ ) - Q(r) Ξ                       (eq:chandra)

W(r) = e^{3Λ-Φ} (ε+p)/r^2                                     (eq:bW)
P(r) = e^{Λ-3Φ} γ p / r^2                                     (eq:bP)
Q(r) = -e^{Λ-3Φ} (ε+p)/r^2 [ Φ'' - (2/r)Φ' - (Φ')^2 ]
        + 4π e^{3Λ-3Φ} (ε+p)^2 / r  ( Φ' - 1/r )              (eq:Qb)
```
with `Ξ ≡ r^2 e^Φ ξ` (line 575), line element `ds^2 = -e^{-2Φ}dt^2 + e^{2Λ}dr^2 + r^2 dΩ^2` (line 493).
Mode expansion `Ξ(t,r) = Σ_j e^{i ω_j t} φ_j(r)` (eq:eigenexpand). **Stable ⇔ all ω_j real ⇔ all ω_j^2 > 0** (line 598). Rayleigh quotient:
```
(ω_j^{(0)})^2 = [ ∫_0^R dr P|∂_r φ|^2 + Q|φ|^2 ] / [ ∫_0^R dr W|φ|^2 ]
```
Inner product: `⟨φ|ψ⟩ = ∫_0^R dr W(r) φ̄ ψ` (line 813).

### 3.2 First-order eigenfrequency correction (Eq. `eq:evcorr`)
```
ω_j^{(1)} = (i/2) ⟨φ_j|F φ_j⟩/⟨φ_j⟩ + (i/2)(1/(ω_j^{(0)})^2) ⟨φ_j|G φ_j⟩/⟨φ_j⟩
```
**Stability rule:** configuration stable to viscosity ⇔ RHS has positive imaginary part, i.e. `Re(i ω_j^{(1)}) < 0` (line 841).

### 3.3 F and G operator definitions (THE F/G operators — Eqs. `eq:fop` and following)
```
W(r) F φ = -∂_r[ (ζ + 4η/3) e^{Λ-2Φ} ∂_r φ / r^2 ]
           + (4η/r) e^{Λ-2Φ} (Λ'-2Φ') φ/r^2  - (4 ∂_r η / r) e^{Λ-2Φ} φ/r^2
           - e^{Λ-Φ} κ T (ε+p)/n ∂_r[ (ε+p)/(nT) (c_s^2-c_n^2) e^{-Φ} ∂_r φ / r^2 ]    (eq:fop)

W(r) G φ = -∂_r( e^{Λ-3Φ} 4π r γ p Y[φ] )
           - (ε+p) e^{Λ-3Φ} 4π (r Φ' - 1) Y[φ]
           - ∂_r[ (e^{-Φ} c_n^2 / r^2) ∂_r( e^{-Λ-2Φ} r^2 Y[φ] ) ]
           + Φ' (e^{-Φ}/r^2) ∂_r( e^{-Λ-2Φ} r^2 Y[φ] )

Y[φ] = -κ T (ε+p)/n ∂_r[ (ε+p)/(nT) (c_s^2-c_n^2) e^{-Φ} ∂_r φ / r^2 ]
```
- `F` = (negative of) last three lines of RHS of Eq. `eq:exp1`; `G` = (negative of) first two lines.
- First two terms of `F` = bulk+shear part; the `κ` term = heat-conduction part of `F`.

### 3.4 BULK + SHEAR stability statement (Sec. IV.A.1, κ=0)
**Result: a star is ALWAYS radially mode stable to bulk and shear viscosity** (lines 871, 1493), proven by F being a positive operator. The key positive-definite identity (Eq. `eq:perfectsquare`):
```
⟨φ_j|F φ_j⟩ = ∫_0^R dr  ζ (e^{Λ-2Φ}/r^2) |∂_r φ_j|^2
              + (4/3) η (e^{Λ-2Φ}/r^2) | ∂_r φ_j - (3/r) φ_j |^2          (eq:perfectsquare)
```
Boundary condition required (vanishing boundary term):
```
lim_{r→R} (ζ + 4η/3)/r^2  ∂_r φ_j = 0     →  (ζ,η) → 0 at least as fast as γp → 0   (lines 858-862)
```
Equivalent energy argument (κ=0): `dE/dt = -2⟨Ẋ|F Ẋ⟩ ≤ 0` with `E = ⟨Ẋ⟩ + ⟨Ξ|C Ξ⟩`; **stable ⇔ both C and F have only positive eigenvalues, for ANY amount of viscosity** (line 929). Important corollary: bulk/shear viscosity **cannot stabilize** a star that is unstable to radial perturbations (line 935).

### 3.5 HEAT-CONDUCTION stability criterion (THE core inequality)

**Necessary-and-sufficient integral condition (Eq. `eq:heatint`, lines 978–989):**
```
Re(i ω_j^{(1)}) = - [ ∫_0^R dr ( L|∂_r^2 φ_j|^2 + M|∂_r φ_j|^2 + N|φ_j|^2 ) ] / [ ∫_0^R dr ( P|∂_r φ_j|^2 + Q|φ_j|^2 ) ]
                  - [ ∫_0^R dr ( K|∂_r φ_j|^2 + H|φ_j|^2 ) ] / [ ∫_0^R dr W|φ_j|^2 ]      (eq:heatint)
```
Stable to (small) heat conductivity ⇔ RHS of `eq:heatint` **negative for all eigenvectors φ_j^{(0)}**.

**High-frequency (necessary) reduction (Eq. `eq:highmodes`, lines 1004):**
```
Re(i ω_j^{(1)}) ~ - [ ∫_0^R dr L|∂_r^2 φ_j|^2 ] / [ ∫_0^R dr P|∂_r φ_j|^2 ]
                 - [ ∫_0^R dr K|∂_r φ_j|^2 ] / [ ∫_0^R dr W|φ_j|^2 ]                     (eq:highmodes)
```
(dominance `|∂_r^2 φ| ≫ |∂_r φ| ≫ |φ|`).

**Sufficient ALGEBRAIC condition for high-frequency stability (Eq. `eq:condi`, lines 1039–1041) — the exact inequality, eq number `eq:condi`:**
```
inf( L/P ) + inf( K/W ) ≥ 0        (eq:condi)        infima over (0,R)
```
This is NOT sufficient for the whole star; low-frequency modes must still be checked.

**L and K functions (lines 1052–1056):**
```
L = -κ ( (ε+p)/n )^2 (e^{-Λ-4Φ}/r^2) c_n^2 [ c_s^2 - c_n^2 ]
K =  κ ( (ε+p)/n )^2 (e^{Λ-2Φ}/r^2)        [ c_s^2 - c_n^2 ]
```
Sign logic (κ ≥ 0), lines 1058–1064:
- `K ≥ 0`  ⇔  `c_n^2 ≤ c_s^2`.
- `L ≥ 0`  ⇔  `c_n^2 [c_n^2 - c_s^2] ≥ 0`, i.e. (with `c_s^2∈[0,1]`) `c_n^2 ≤ 0` OR `c_n^2 ≥ c_s^2`.

**Summary stability conditions (Conclusions, lines 1497–1501):**
- *Necessary* condition: at no point in the star can BOTH `c_n^2[c_s^2-c_n^2] ≤ 0` AND `c_s^2 - c_n^2 ≥ 0` be violated → **automatically satisfied whenever `c_s^2 ∈ [0,1]`**.
- *Sufficient* condition (i): `c_n^2 ≤ 0` everywhere in the star.
- *Sufficient* condition (ii): `inf(L/P) + inf(K/W) ≥ 0` [Eq. `eq:condi`].
- If either sufficient condition holds, only a finite number of low-frequency modes need checking.

**K, H, O coefficients from the heat part of F (App. B, lines 1706–1708):**
```
K = e^{Λ-Φ} κ ( (ε+p)/n )^2 (c_s^2 - c_n^2) e^{-Φ}/r^2
O = -∂_r( e^{Λ-Φ} κ T (ε+p)/n ) (ε+p)/(nT) (c_s^2-c_n^2) e^{-Φ}/r^2
H = (1/2) ∂_r[ ∂_r( e^{Λ-Φ} κ T (ε+p)/n ) (ε+p)/(nT) (c_s^2-c_n^2) e^{-Φ}/r^2 ]
```
(Note: the body's `K` in `eq:heatint` and App. B's `K` differ by an `e^{2Λ-2Φ}` weighting convention — body K carries `e^{Λ-2Φ}`, App. K carries `e^{Λ-2Φ}` after combining `e^{Λ-Φ}·e^{-Φ}`; they are the same object. Reproduce App. B forms.)
`N = 0` for G (a "happy coincidence", line 1728). `L` (App. B, line 1731) matches body. `M, E, F` for G "refrained from explicitly writing out... due to their large size" (line 1768) — **NOT given in the paper.**

Boundary conditions for heat operators to be (anti)self-adjoint (lines 1711–1725):
```
lim_{r→R} κ (c_s^2-c_n^2) ∂_r φ / r^2 = 0    (eq:cond)
∂_r κ (c_s^2-c_n^2) → 0  at r=R
κ (c_s^2-c_n^2) → 0 at least as fast as γp → 0 at the boundary
```

### 3.6 BDNK FAST-TIMESCALE oscillations (speed of second sound) — closed-form eigenfrequencies

Telegrapher system (Eq. `eq:tele`, fast time `τ=t/α`, rescaled radius `ρ=r/α`, lines 1291–1294):
```
∂_τ ΔẼ^{(0)}   = -(e^{-Φ}/τ_E) ΔẼ^{(0)} - e^{-2Λ-Φ} ∂_ρ ΔQ̃_r^{(0)}     (eq:telee)
∂_τ ΔQ̃_r^{(0)} = -(e^{-Φ}/τ_Q) ΔQ̃_r^{(0)} - e^{-Φ}(τ_P/τ_E) ∂_ρ ΔẼ^{(0)}  (eq:teleq)
```
Decoupled wave eq (Eq. `eq:telefull`):
```
[ ∂_τ^2 - e^{-2Λ-2Φ}(τ_P/τ_E) ∂_ρ^2 ] ΔẼ^{(0)}
  + e^{-Φ}[ 1/τ_E + 1/τ_Q ] ∂_τ ΔẼ^{(0)} + (e^{-2Φ}/(τ_E τ_Q)) ΔẼ^{(0)} = 0
```
**Eigenvalues** (BCs `∂_ρψ_E(0)=0`, `ψ_E(R/α)=0`):
```
λ_j = ( (j+1/2) π α / R )^2,   j = 0,1,2,...
ψ_{j,E}(ρ) = cos(√λ_j ρ)
```
**Closed-form fast-timescale eigenfrequencies (Eq. `eq:fasteig`, lines 1345–1348) — exact target:**
```
Ω_j^± = (i/2) e^{-Φ} [ 1/τ_E + 1/τ_Q ]  ±  e^{-Φ} √( e^{-2Λ} c_τ^2 λ_j - (1/4)[ 1/τ_E - 1/τ_Q ]^2 )
```
**Speed of second sound (Eq.):** `c_τ^2 = τ_P / τ_E` (line 1330).
Speed of propagation in fast timescale = `e^{-Λ-Φ} √(τ_P/τ_E)`; compare Chandrasekhar speed `√(P/W) = e^{-Λ-Φ} c_s`.
**Causality constraint:** `τ_P < τ_E` (equivalently `τ_E ≥ τ_P`), matching BDNK flat-space condition Eq. (21d) of Bemfica2022 (lines 1332–1333, 1513).
Most-unstable limit (`α/R→0`): `Ω_0^+ = i e^{-Φ}/τ_E`, `Ω_0^- = i e^{-Φ}/τ_Q` (Eq. `eq:fulldecay`) → stability ⇔ `τ_E>0, τ_Q>0` (hence `τ_P>0` since `c_τ^2>0`). **All fast modes decaying → never an instability.**

## 5. Reproduction method (concrete)

Since there are no numeric targets, "reproduction" = symbolic/numeric verification of the analytic claims:
1. Integrate TOV for a chosen 1D EOS `p(ε)` (e.g. polytrope or Mathews ideal gas) → background `Φ(r), Λ(r), ε(r), p(r), n(r)`.
2. Solve the perfect-fluid Chandrasekhar Sturm–Liouville problem `eq:chandra` with `W,P,Q` (`eq:wpq`) → eigenpairs `(ω_j^{(0)}, φ_j^{(0)})`, BC `γp/r^2 ∂_rφ → 0` at R.
3. **Verify bulk+shear:** assemble `⟨φ_j|F φ_j⟩` two equivalent ways (`eq:stablvis` raw and `eq:perfectsquare` sum-of-squares); confirm positivity for `ζ,η ≥ 0`.
4. **Verify heat criterion:** compute `L, K` (and `M,N,H,O`) from §3.5, form `eq:heatint`/`eq:highmodes`; check sign; test the algebraic `eq:condi` `inf(L/P)+inf(K/W) ≥ 0` and the `c_n^2 ≤ 0` sufficient condition. For Mathews gas confirm `L≥0, K≤0`.
5. **Verify BDNK fast modes:** plug into telegrapher system, confirm `Ω_j^±` (`eq:fasteig`), `c_τ^2=τ_P/τ_E`, all-decaying property, causality `τ_P<τ_E`.

Tolerances: analytic; match equations symbolically. Any numeric eigenvalue check is solver-defined, not paper-defined.
